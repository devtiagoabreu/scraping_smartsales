# Sistema de Web Scraping DGB COMEX

Sistema profissional para coleta automatizada de dados de estoque da DGB COMEX.

## Funcionalidades

- ✅ Login automático no sistema DGB
- ✅ Consulta de estoque por produto
- ✅ Processamento em lote de múltiplos produtos
- ✅ Exportação para CSV individual por produto
- ✅ Consolidação de todos os dados em um único arquivo
- ✅ Geração de relatórios PDF
- ✅ Interface web responsiva
- ✅ Monitoramento em tempo real
- ✅ Logs detalhados de execução
- ✅ Screenshots automáticos para debugging
